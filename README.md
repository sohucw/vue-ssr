# vue-ssr-tech

## Build Setup

``` bash
# 安装依赖包
npm install

# 客户client端服务热重载 at localhost:8000
npm run dev:client

# 服务server端服务热重载 at localhost:3333
npm run dev:server

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
